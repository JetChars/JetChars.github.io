PEP8¶

Python Enhancement Proposal [1] 8 is a Style guide for python code [2]

Pylint¶

pylint is a python code analysing tool - can analys faults in python code. - find codes ummet pep8

pylint in vim¶

set makeprg=pylint\ --reports=n\ --output-format=parseable\ %:p
set errorformat=%f:%l:\ %m

autocmd FileType python set makeprg=pylint\ --reports=n\ --msg-template=\"{path}:{line}:\ {msg_id}\ {symbol},\ {obj}\ {msg}\"\ %:p
autocmd FileType python set errorformat=%f:%l:\ %m
  • add either of above config to /etc/vim/vimrc
  • exec pylint by using cmd :make within vim
